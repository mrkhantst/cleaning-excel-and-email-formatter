# Cleaning Excel and Email Formatter

This file can be clean an excel file to remove duplicate rows, missing rows, removing spaces and will format the email column to make it easy for email sending.

### How to use it

1. Create a folder with your Excel file
2. Change your Excel file name to data.xlsx
3. Change the email column name to Email Address
4. Add this file to your created folder
5. Read the instruction in the file
6. Run the file
